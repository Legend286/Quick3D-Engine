# `Editor/`

Avalonia application shell. Hosts:

- Docked panels: Viewport, AssetBrowser, Hierarchy, Inspector, Console, Profiler.
- ImGui overlay in the viewport with ImGuizmo.
- Material editor, model viewer, particle editor, level editor, cinematic timeline editor.
- Source-control integration (Git LFS lock/unlock badges).

Built with `dotnet run --project Editor`.
